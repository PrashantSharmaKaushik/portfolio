# Tech Stack

## Core Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion

## Optional UI Libraries

- React Icons
- Lenis (smooth scrolling)
- GSAP (advanced animations)

## No Backend Required

This project is a fully static frontend application.

Do NOT use:

- Node.js backend
- Express.js
- MongoDB
- PostgreSQL
- MySQL
- Firebase
- Supabase
- AWS Services
- Docker
- Authentication Systems
- REST APIs
- GraphQL
- CMS Integrations

All content should be stored as static JavaScript/JSON objects inside the project.

Example:

src/data/
├── experience.js
├── projects.js
├── skills.js
└── profile.js

# Architecture

The website must be a static Single Page Application (SPA).

Pages/Sections:

- Home
- About
- Experience
- Skills
- Projects
- Contact

No routing is required unless explicitly needed.

All sections should exist on a single scrolling page with smooth navigation.
